# Academy

Academy - demo project.

Built with:

- AWS
  - EKS
  - NLB
- Kubernetes (1.18)
- NodeJS (v14)
  - Express
- Python (3.8)
  - FastAPI
- Go (1.16)
  - Fiber

### Application Structure

```
.
├── api               # All our microservices
│   ├── account       # Account microservice with NodeJS
│   ├── comments      # Comments microservice with Go
│   └── courses       # Courses microservice with Python
├── kubernetes
├── web
├── LICENSE
└── README.md
```
